# Welcome to the Demo Vault

This is a **Retro Obsidian Publish** demo vault — a collection of interconnected notes rendered with a monochrome Macintosh System 1 aesthetic.

> "The purpose of a note-taking system is not to store information, but to think with it." — Niklas Luhmann

## How to Navigate

Use the **sidebar** on the left to browse notes by folder. Click any `[[wiki link]]` to jump between notes. The **backlinks panel** on the right shows which notes point to the current one. Toggle the **graph view** from the menu bar to see the full connection map.

Press `/` at any time to open the search bar.

## What's in this Vault

This vault is organised into three clusters:

### Philosophy

- [[Stoicism]] — the ancient school of virtue ethics
- [[Epistemology]] — the theory of knowledge and justified belief
- [[Zeno of Citium]] — founder of the Stoic school
- [[The Examined Life]] — Socrates and the practice of self-examination

### Computing

- [[The Zettelkasten Method]] — Luhmann's slip-box system for knowledge management
- [[Personal Knowledge Management]] — modern PKM tools and workflows
- [[Unix Philosophy]] — the design principles behind Unix and its descendants
- [[Hypertext]] — the history and theory of linked documents

### Writing

- [[How to Write Clearly]] — principles of clear, direct prose
- [[The Writing Process]] — from first draft to final revision
- [[Note-Taking Strategies]] — methods for capturing and processing ideas

## Markdown Features Demonstrated

This vault exercises the full range of Markdown features supported by the renderer:

- **Headings** (H1–H4), **bold**, *italic*, ~~strikethrough~~
- `inline code` and fenced code blocks with syntax highlighting
- Blockquotes with attribution
- Tables with header rows
- Ordered and unordered lists, nested lists
- Task lists
- Horizontal rules
- [[wiki links]] with aliases like [[Stoicism|Stoic philosophy]]
- Broken links like [[Nonexistent Note]] (shown in red)
